About

The Smart Attendance System automates student attendance using AI-based face recognition, eliminating manual roll calls. Built with Python, Flask, OpenCV, and the face_recognition library, it detects and identifies students from classroom images or live video in real time. Each student’s face is encoded into numerical vectors, which the system compares against a pre-trained dataset to mark attendance accurately. Attendance records are stored in CSV or Excel files for easy management. This system saves time, reduces errors, prevents fraudulent marking, and provides a scalable, user-friendly solution for schools, colleges, and training centers.
